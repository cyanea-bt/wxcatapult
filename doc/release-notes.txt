Release Notes for openMSX Catapult 0.5.2-R1 (2005-06-18)
========================================================

This new release brings the following improvements:

User interface:

Fixed: The rensha Turbo slider wasn't updated when updated in openMSX.
Fixed: Choosing a printer log file works now as it should.
Fixed: In Windows, it was possible to crash openMSX by using F10 or ALT

Compilation:

Added: Experimental and untested support for the following CPU's:
	DEC Alpha
	ARM
	HP PA-RISC
	IA-64
	Motorola 680x0
	MIPS
	IBM S/390
	Sparc

Known issues / caveats:
- comboboxes behave a bit strangely sometimes on non-Windows systems
- the select extension listbox doesn't work properly when using wxWidgets 2.5.x
  Use the spacebar to deselect extensions
- typing into openMSX doesn't support language specific characters yet
- still not complete
