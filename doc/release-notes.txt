Release Notes for openMSX Catapult for openMSX 19.0-RC1 (2023-07-12)
====================================================================

Catapult release to match openMSX 19.0.

This release we again didn't change much, but is the full list of news:
- migrated to wxWidgets 3.2.1 and libxml2 2.9.14 (latest stable releases)
- adjusted code for changes in the openMSX type command

Known issues / caveats:
- not feature complete, especially for newer features you will have to revert
  to the openMSX console (press F10/Cmd+L) or the OSD menu (MENU key/Cmd+O)

openMSX Home Page:
  http://openmsx.org/
Project page on GitHub:
  http://github.com/openMSX/
